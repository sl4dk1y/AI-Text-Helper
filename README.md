# Название проекта
AI Text Helper

## Описание
Этот проект помогает работать с текстами. Он умеет:
- Исправлять ошибки
- Делать текст короче
- Находить главные мысли

## Кто будет пользоваться
- Писатели
- Студенты
- Все, кто работает с текстами

## Какие программы использую
- Python (язык программирования)
- FastAPI (для создания API)
- Ollama (для нейросети)

## Почему я выбрал эту тему
Я выбрал помощника для текстов, потому что это полезно. Многие люди пишут тексты и делают ошибки. Мой проект поможет их исправлять.

## Как это будет работать
1. Человек отправляет текст на сайт
2. Мой сервер получает текст
3. Сервер отправляет текст нейросети
4. Нейросеть исправляет ошибки
5. Исправленный текст возвращается человеку

## Какие трудности могут быть
- Нейросеть может думать долго
- Может сломаться интернет
- Может прийти слишком большой текст

## Как я буду решать трудности
- Сделаю так, чтобы сервер не зависал
- Буду ловить ошибки и сообщать о них
- Ограничу длину текста
